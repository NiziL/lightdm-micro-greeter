# lightdm-micro-greeter
A minimalist LightDM greeter written in go, powered by [gotk3](github.com/gotk3/gotk3)

## Features
- multi-user login
- no user list 
- only starts the default session
- must have a file /etc/lightdm/wallpaper.png
